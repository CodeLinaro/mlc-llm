MLC-LLM for Adreno™ GPU documentation
=====================================

Welcome to documentation of MLC-LLM for Adreno™ GPU targets.

Adreno™ is a series of graphics processing unit (GPU) semiconductor intellectual
property cores developed by Qualcomm and used in many of their SoCs.
The  Adreno™ GPU accelerates the rendering of complex geometries to deliver high-performance
graphics and a rich user experience with low power consumption.

This documentation details about open source solutions for running various Large Language Models
on Adreno GPU with best performance possible.
